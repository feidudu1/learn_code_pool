# python-data-science

课程地址：慕课网
本课程会教大家如何使用 Python 分析和可视化数据，开启 Data Science 职业之旅。

本课程是 Python for Data Science 的入门课程，是学习 Text Mining，Machine Learning，Deep Learning 的基础。

主要包含以下內容：

- anaconda 和 Jupyter 的介绍和安装
- numpy 入门
- pandas 入门
- 使用 numpy+pandas 进行数据分析和处理
- 数据可视化：matplotlib 入门
- 项目实战
